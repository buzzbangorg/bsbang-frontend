Buzzbang

Google-like search engine for biological data

Buzzbang is a Google-like search engine for Life Sciences data marked up with bioschemas.org schemas. The data from different resources is collated and made searchable via this web interface. The project is in its early stages since the markup is evolving and data resources are just beginning to adopt it. If you're interested in adding bioschemas.org markup to your data source, please see the bioschemas.org website and technical pages.

Resources collated

This is the first stable version of Buzzbang. Therefore, at the moment we have indexed only Biosamples published by European Bioinformatics Institute (EMBL-EBI) . We are working on scraping and indexing more data that have been publised with bioschemas.org mark-up.

For Developers

We welcome contributions to Buzzbang from developers and enthusiasts, whether involved in science or not. For further details please visit the project wiki. This has information on the architecture and links to the different project components.
