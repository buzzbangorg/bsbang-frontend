Buzzbang

Google-like search engine for biological data

Buzzbang is a Google-like search engine for Life Sciences data marked up with bioschemas.org schemas. The data from different sources is collated and made searchable via this web interface. The project is in its early stages since the markup is evolving and data sources are just beginning to adopt it. If you're interested in adding bioschemas.org markup to your data source, please see the bioschemas.org website and technical pages.

For Developers

This is the first stable version of Buzzbang that has been released. Presently, we have indexed only Biosamples published by European Bioinformatics Institute (EMBL-EBI) . We are working on to scrape and index more data that have been publised with Bioschemas mark-up. Developers and enthusiasts are welcomed to contribute to this open-source project. For further details please visit our Github repository.
